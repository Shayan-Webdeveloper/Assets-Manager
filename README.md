# 🚀 MaintainIQ

**MaintainIQ** is a modern web-based **Asset Maintenance & Issue Tracking System** that helps organizations manage their assets, report issues, monitor maintenance history, and improve equipment reliability. The application features a clean dashboard, QR-based asset identification, AI-powered maintenance suggestions, and secure user authentication.

---

# 📌 Features

### 🔐 Authentication

* Secure login using Supabase Authentication
* Role-based access (Administrator & Technician)
* Session management
* Protected dashboard routes

### 📊 Dashboard

* Total registered assets
* Open issues count
* Critical priority issues
* Resolved issues this month
* Recent maintenance activity timeline
* Assets requiring immediate attention

### 🗂️ Asset Management

* Register new assets
* View asset details
* Update asset information
* Asset status tracking
* QR Code generation for every asset
* Asset maintenance history

### 🛠️ Issue Management

* Report maintenance issues
* Assign issue priority
* Track issue status
* View issue history
* Resolve and close issues

### 🤖 AI Maintenance Suggestions

* Analyze reported problems
* Suggest possible causes
* Recommend maintenance actions
* Display estimated downtime and required tools

### 📱 Responsive Design

* Desktop-friendly dashboard
* Mobile responsive layout
* Collapsible sidebar for smaller screens

---

# 🔄 Workflow

1. User signs in.
2. Administrator registers assets.
3. QR code is generated for each asset.
4. Technician scans or opens the asset.
5. Issues are reported.
6. AI provides maintenance suggestions.
7. Maintenance history is automatically recorded.
8. Dashboard statistics update in real time.
